A 类 · 深度用户
2/12
Champion 候选池
判定标准
  • 近 30 天 ≥2 个高价值场景持续使用
  • 有 Prompt 库或工作流笔记
  • 有分享意愿
动作
  • 每双周主持一次道场
  • 每季度交付 1 个场景级资产
  • 担任资产库主评审
激励
  • 思南半年大会颁发认证
  • 部门 OKR 加权
  • 优先参加公司级 AI 项目
B 类 · 一般用户
7/12
战例人 / 共创人
判定标准
  • 会用 AI 但停在单点场景
  • 未形成工作流
  • 单点场景如润色 / 格式 / 翻译
动作
  • 每季度 ≥1 次担任战例提供人
  • 参与 ≥2 次资产共创
  • 复用资产库内容到自己的方案
激励
  • 战例人纳入季度评价
  • 优秀资产共创公开表彰
  • 积分满后晋升 A 类
C 类 · 浅层用户
3/12
裁判 / 旁观者
判定标准
  • 知道 Claude Code 偶尔尝试
  • 遇到不熟的场景退回传统
  • Prompt 未写顺过
动作
  • 道场担任裁判组成员
  • 从低门槛场景先入门
  • 一个月后升 B 类
激励
  • 低门槛场景上线即获"AI 新兵"认定
  • 升 B 类进入主流通道
  • 无考核压力的起步窗口
↕ 三档不是固化标签 · 流动是激励机制的一部分
C 类→B 类（1 个月）
B 类→A 类（半年）
A 类→B 类（连续 2 届不达标）
